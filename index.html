Mengenal Semantic Element di HTML 5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artiker-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer. karena tag article memiliki type block, artinya element ini memiliki lebar sesuai lebar browser dika tidak menentukan lebarnya.

  • Aside

    • Ketika ini bisa kita gunakan untuk melengkapi layout utama. Misal kita emiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena tag aside mimiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
